# the rogue orchestra player

Static React + Vite music player for a public-but-unlisted GitHub Pages release.

## Local use

```sh
npm install
npm run dev
```

Production check:

```sh
npm run build
npm run preview
```

Tests:

```sh
npm test
```

## Safety

- No backend, accounts, cookies, analytics, telemetry, or API keys.
- Audio and credits are local static assets/data.
- `noindex` metadata and `robots.txt` are included.
- Public-but-unlisted still means anyone with the link can listen.

Deployment setup is intentionally left for later.
